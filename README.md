

## インストール方法
1. fileをダウンロードし，ローカルに解凍してください．
2. 拡張機能を開きます．chrome://extensions/
3. `Developer mode`を有効にします．。
4. `パッケージ化されていない拡張機能を読み込む`をクリックして，`src`というフォルダを選択します．

## 使用方法
Chromeの設定を行います．
```
Chrome settings:
 > Devices:
  > Add custom device...
    > Width: 1366
    > Height: 768
```

ボタンをクリックすると．テーマ用のサムネイル画像が保存されます．
